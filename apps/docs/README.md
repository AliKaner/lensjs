# lensjs docs

Documentation and sandbox site for [`@alikaner/lensjs`](https://www.npmjs.com/package/@alikaner/lensjs). Vite + React, no framework magic.

```bash
npm run dev       # local dev server
npm run build     # builds the package first, then the site into dist/
```

The site uses the package through the workspace (`@alikaner/lensjs: "*"`), so the sandbox always demos the code that actually ships.
